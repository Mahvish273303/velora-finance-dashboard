# Velora – Finance Dashboard

Velora is a modern fintech dashboard web application designed to help users track financial activity, analyze spending behavior, and manage transactions through a clean, premium SaaS-style interface.

This project focuses on frontend architecture, UI/UX excellence, and product thinking without backend complexity.

## 🧠 Project Overview

Velora delivers a production-style financial dashboard experience, emphasizing clarity, interactivity, and scalable frontend design.

It simulates real-world fintech features while maintaining a lightweight and efficient architecture.

---

## ✨ Features

- **Dashboard Overview**  
  Real-time summary cards for total balance, income, and expenses

- **Interactive Charts**  
  Line chart for balance trends and pie chart for category breakdown

- **Transactions Management**  
  Add, delete, filter, search, and sort transactions

- **Role-Based UI**  
  Admin and Viewer modes with dynamic UI behavior

- **Insights Section**  
  Smart financial insights and month-over-month comparisons

- **Export Functionality**  
  Download transactions as CSV

- **Recent Activity Feed**  
  Tracks user actions like transactions and role changes

- **Dark Mode**  
  Smooth and visually appealing theme switching

- **Responsive Design**  
  Optimized across desktop, tablet, and mobile devices

---

## 🛠 Tech Stack

- **React (Vite)**
- **Tailwind CSS**
- **Zustand** (State Management)
- **Recharts** (Data Visualization)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)
